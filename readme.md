#Afroelixir Landing page
###**Getting started**
###Technologies Used
####HTML5, CSS3, Bootstrap 3.3, JavaScript, jQuery
 - Run :  `git clone https://github.com/getwebem/afroelixir.git`
 - Run :  `cd afroelixir`
 - Run :  `Brackets .`
 - Open:  `Index.html`
 - Open:  `Live Preview`
